# Ruthless Chess

A high-stakes, fast-paced chess platform.

## Features

- **Ruthless Speed**: 25s Blitz matches with 1s increment.
- **Premium Tier**: Access strictly guarded Duels and Tournaments ($5.99/mo).
- **Hardcore Ranking**: Calibrate against our Ruthless AI to earn your rank.

## Tech Stack

- React + Vite
- TypeScript
- Chess.js (Logic)
- React-Chessboard (UI)
- Lucide React (Icons)
- Vanilla CSS Variables (Design System)

## Getting Started

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Design System

- **Primary**: Ruthless Red (`#dc2626`)
- **Background**: Void Black (`#050505`)
- **Font**: Inter/System
